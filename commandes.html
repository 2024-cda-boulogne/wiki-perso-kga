Les commandes git les plus utiles :

"Git init"

Cette commande sert à initialiser son projet en local afin de la déployer sur Github par la suite

"Git add ."

Cette commande sert à sélectionner les fichiers voulu pour ensuite les envoyer sur Github avec d'autres commandes

"Git commit -m "Oui"

Cette commande sert à préparer à l'envoi les fichiers sélectionnés avec la commande "git add ." pour les envoyer par la suite

"Git push"

Cette commande sert à envoyer les fichiers sur Github

"Git pull"

Cette commande sert à récuperer un projet hébergé sur Github
